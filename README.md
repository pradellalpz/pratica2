# SecureChain Audit

**Plataforma de Auditoria Baseada em Blockchain**
Disciplina: Segurança de Sistemas com Blockchain, Criptografia e Auditoria de Eventos

Projeto acadêmico para a empresa fictícia *SecureData Solutions*, desenvolvido em
**Python 3** e **Bash** para **Linux Debian**.

> 🚧 Estrutura inicial do projeto. Documentação de execução e capturas serão
> adicionadas conforme os módulos forem implementados.
